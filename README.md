# 📊 Customer Behavior Analysis Dashboard (Power BI)

## 📌 Overview

This project showcases an interactive **Power BI dashboard** designed to analyze customer behavior and business performance. It helps in identifying patterns, understanding customer trends, and making data-driven decisions.

---

## 🎯 Objectives

* Analyze customer purchasing behavior
* Identify key customer segments
* Track sales and revenue trends
* Provide actionable business insights

---

## 📊 Dashboard Features

* Customer segmentation analysis
* Sales and revenue visualization
* Trend analysis over time
* KPI indicators for performance tracking
* Interactive filters for better insights

---

## 🛠️ Tools & Technologies

* **Power BI** – Data visualization and dashboard creation
* **DAX** – Measures and calculated columns
* **Data Modeling** – Relationship building between datasets
